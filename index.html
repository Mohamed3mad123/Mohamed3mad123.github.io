Digital Marketing & Social Media Insights

📌 Project Overview

A comprehensive end-to-end data analysis project for an ice cream retail business (2023 - 2025). The project transforms raw sales data into interactive dashboards to monitor performance and simulate marketing decisions.

❓ Business Problem

  • Difficulty identifying profitable products
  • Unclear peak hours
  • Discount impact not measurable

🧹 Data Cleaning & Preparation

  • Power Query ETL
  • Calendar Table
  • Data Modeling
  • DAX Measures

📈 Analysis & Insights

  • Time Intelligence
  • Branch Comparison
  • Market Basket Analysis
  • What-if Analysis

⚙️ Tools & Skills

Power BI - Excel - DAX - SQL

💡 Key Insights

  • Peak at 11 PM
  • Friday highest sales
  • Waffles highest margin

🚀 Recommendations

  • Increase staff at peak
  • Create combo offers
  • Focus high-margin products
